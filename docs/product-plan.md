# FixPic 产品计划书

## 一、产品定位

### 1.1 核心定位
**电商卖家的一站式 AI 图片处理工具**

不是通用图片工具，而是专门为电商卖家优化的场景化解决方案。

### 1.2 目标用户
- 淘宝/天猫卖家
- 京东卖家
- 拼多多卖家
- 跨境电商卖家（Amazon、eBay、速卖通）
- 小红书/抖音电商卖家
- 独立站卖家（Shopify）

### 1.3 核心价值主张
> "一键出图，多平台适配，比 Photoshop 快 100 倍"

## 二、市场调研

### 2.1 电商卖家痛点

| 痛点 | 描述 | 严重程度 |
|------|------|:--------:|
| **效率低** | 单张产品图处理需要 10-30 分钟（PS 手工） | ★★★★★ |
| **成本高** | 外包抠图 ¥1-5/张，批量成本高 | ★★★★☆ |
| **规范复杂** | 不同平台尺寸、白底图要求不同 | ★★★★☆ |
| **质量不稳定** | 手工处理质量参差不齐 | ★★★☆☆ |
| **批量需求** | 新品上架需批量处理 50-200 张图 | ★★★★★ |

### 2.2 各平台图片规范

| 平台 | 主图尺寸 | 白底图要求 | 文件限制 |
|------|----------|------------|----------|
| 淘宝/天猫 | 800×800px | 第5张必须白底 | ≤3MB, JPG |
| 京东 | 800×800px | 首图必须纯白底 | ≤2MB |
| 拼多多 | ≥480×480px | 主图纯白底（服饰除外） | ≤3MB |
| Amazon | 2000×2000px（推荐） | 纯白 RGB(255,255,255) | ≤10MB |
| eBay | 800-1600px | 白到浅灰均可 | - |
| Shopify | ≥800px（缩放需要） | 建议纯色背景 | - |

### 2.3 竞品分析

| 产品 | 优势 | 劣势 | 定价 |
|------|------|------|------|
| **PhotoRoom** | 移动端体验好，电商场景优化 | API 价格较高 | $12.99/月 |
| **Remove.bg** | 品牌知名度高，效果稳定 | 功能单一，只能抠图 | $0.20/张 |
| **Claid.ai** | API 功能全面，批量处理强 | 学习成本高 | $15/月起 |
| **佐糖** | 中文友好，价格低 | AI 效果一般 | ¥9.9/月 |
| **稿定设计** | 模板丰富，电商适配好 | 需要设计基础 | ¥99/年 |

## 三、产品规划

### 3.1 核心功能矩阵

```
电商产品图处理流程：

原图（有水印/杂乱背景）
    │
    ├─→ [1] 去水印（Dewatermark.ai）
    │
    ├─→ [2] AI 抠图（Replicate BRIA）
    │
    ├─→ [3] 换背景
    │       ├─ 纯白/纯色背景
    │       ├─ 渐变背景
    │       ├─ AI 生成场景背景
    │       └─ 自定义背景图
    │
    ├─→ [4] 图片增强
    │       ├─ 4X 放大
    │       ├─ 锐化/降噪
    │       └─ 自动调光
    │
    ├─→ [5] 尺寸适配
    │       ├─ 淘宝 800×800
    │       ├─ 京东 800×800
    │       ├─ 拼多多 480×480+
    │       ├─ Amazon 2000×2000
    │       └─ 自定义尺寸
    │
    └─→ [6] 批量导出
            ├─ 压缩优化
            ├─ 格式转换
            └─ 打包下载
```

### 3.3 换背景 API 对比

| 方案 | 优势 | 劣势 | 价格 | 推荐度 |
|------|------|------|------|:------:|
| **本地 Canvas** | 免费、快速、纯色/渐变 | 不支持 AI 生成 | 免费 | ★★★★★ |
| **PhotoRoom API** | 效果好、电商优化 | 价格高 | $0.20/张 | ★★★★☆ |
| **Replicate SDXL** | 灵活、可控 | 需要 prompt 调优 | ~$0.05/张 | ★★★★☆ |
| **Claid.ai** | 功能全面 | 学习成本 | ~$0.02/张 | ★★★☆☆ |

**建议：**
- 纯色/渐变背景 → 本地 Canvas（免费）
- AI 场景背景 → Replicate SDXL-AD-Inpaint（性价比高）

## 四、场景化优化

### 4.1 一键出图流程

```
用户上传原图
    ↓
[自动识别] 是否有水印？
    ├─ 是 → 自动去水印
    └─ 否 → 跳过
    ↓
[自动抠图] AI 移除背景
    ↓
[选择背景]
    ├─ 纯白（淘宝/京东默认）
    ├─ 渐变
    ├─ AI 生成场景
    └─ 自定义上传
    ↓
[选择平台] 自动调整尺寸
    ├─ 淘宝 800×800
    ├─ 京东 800×800
    ├─ 拼多多 750×750
    ├─ Amazon 2000×2000
    └─ 自定义
    ↓
[自动优化] 压缩到平台限制内
    ↓
[导出] 单张下载 / 批量打包
```

### 4.2 电商平台预设模板

| 模板名称 | 尺寸 | 背景 | 格式 | 大小限制 |
|----------|------|------|------|----------|
| 淘宝主图 | 800×800 | 白底 | JPG | ≤3MB |
| 淘宝白底图 | 800×800 | 纯白 | JPG | 38K-300K |
| 京东主图 | 800×800 | 白底 | JPG | ≤2MB |
| 京东透明图 | 800×800 | 透明 | PNG | ≤1MB |
| 拼多多主图 | 750×750 | 白底 | JPG | ≤3MB |
| 拼多多活动图 | 750×352 | 白底居中 | JPG | ≤3MB |
| Amazon 主图 | 2000×2000 | RGB(255,255,255) | JPG | ≤10MB |
| 小红书封面 | 1242×1660 | 自由 | JPG | - |

### 4.3 批量处理功能

- **批量上传**：拖拽文件夹，一次上传 100+ 张
- **统一设置**：选择目标平台，批量应用模板
- **流水线处理**：去水印 → 抠图 → 换背景 → 调尺寸 → 压缩
- **进度追踪**：实时显示处理进度
- **打包下载**：ZIP 压缩包一键下载

## 八、总结

### 核心策略
1. **场景聚焦**：专注电商卖家，不做大而全
2. **API 套壳**：集成最优 API，不重复造轮子
3. **体验优化**：流程简化，一键完成多步操作
4. **成本控制**：本地处理为主，AI 处理为辅

### 差异化优势
1. 平台规范预设（自动适配淘宝/京东/拼多多）
2. 一键处理流水线（vs 其他工具需要多步操作）
3. 批量处理能力（vs 竞品单张处理）
4. 中国电商场景优化（vs 国外工具水土不服）

### 下一步行动
1. 测试 Replicate SDXL-AD-Inpaint 换背景效果
2. 开发换纯色背景功能
3. 添加电商平台尺寸预设
4. 实现批量处理流水线
